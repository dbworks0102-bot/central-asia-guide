# SEOキーワードキュー

週次の記事自動下書き生成（`scripts/weekly-draft.ps1`）が参照する唯一のキーワード管理台帳。
生成側は未使用（チェックなし）の項目から1つ選び、下書き（`status: "draft"`）を
`src/data/articles.js` に追記した後、対応する項目にチェックを入れて `使用済み` と
記事の `slug` を追記すること。人間が手動で書く場合も同じ運用でよい。
